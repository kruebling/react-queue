# react-queue
This website follows the Epicodus tutorials to create a help queue in React.

## Getting Started

  1. `cd Desktop`
  2. `git clone https://github.com/kruebling/react-queue.git`
  3. `cd react-queue`
  4. `npm install`
  4. `npm serve`
